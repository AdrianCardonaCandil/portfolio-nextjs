# **App Name**: Click Counter

## Core Features:

- Counter Display: Display the current count on the landing page.
- Increment Button: A clickable button to increment the counter by one.
- Greeting Message: Display a welcome message on the landing page.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) for a calm and inviting feel.
- Background color: Very light blue (#F0F8FF) to provide a clean backdrop.
- Accent color: Soft green (#90EE90) for button and interactive elements to add a touch of vibrancy.
- Body and headline font: 'PT Sans', a humanist sans-serif suitable for both headlines and body text, giving a modern and slightly warm appearance.
- Simple and centered layout for the greeting message, counter, and button.
- Subtle animation or highlighting effect on the button click to provide visual feedback.
